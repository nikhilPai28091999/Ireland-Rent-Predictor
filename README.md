      IRELAND RENT PREDICTOR 

1. Built an Ireland Rent Predictor web app with React and FastAPI to estimate rental prices based on user inputs (bedrooms, bathrooms and location)
2. Designed a Random Forest regression model trained on scraped & cleaned rental data from daft.ie to provide accurate rent predictions
3. Integrated an intelligent Rasa AI chatbot interface for conversational queries
4. Dockerized the project for seamless deployment
